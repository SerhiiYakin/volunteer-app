# Volunteer App

Volunteer App is a full-stack system consisting of backend, web frontend,
and mobile application.

## Project Architecture

This project is split into multiple repositories to keep concerns separated:

### Backend
- Java / Spring Boot
- REST API
- Repository: https://github.com/SerhiiYakin/volunteer-app-server

### Web Frontend
- JavaScript
- Repository: https://github.com/SerhiiYakin/volunteer-app-frontend

### Mobile App
- JavaScript
- Repository: https://github.com/SerhiiYakin/volunteer-app-mobile

## Project Goal

This project was developed as part of my Bachelor's degree requirements
and focused on applying theoretical knowledge to a practical, real-world
software system.
